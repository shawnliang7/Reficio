# Reficio GitHub + Vercel Deployment Guide

## 1. Import The Repo Into Vercel

1. Go to https://vercel.com/new
2. Choose **Import Git Repository**
3. Select the GitHub repository named `Reficio`
4. Use these settings:
   - Framework Preset: Other
   - Build Command: empty
   - Output Directory: empty
   - Install Command: empty
5. Click **Deploy**.

## 2. Permanent Link

After deployment, Vercel will give a link similar to:

```text
https://reficio.vercel.app
```

You can rename the project slug in Vercel if the exact `reficio.vercel.app` name is unavailable.

## 3. Link-Only Viewing

For a school presentation, the simplest option is to keep the Vercel link unlisted and only share it with judges or classmates.

If you need password protection, open the Vercel project settings and check whether Deployment Protection is available on your plan.
